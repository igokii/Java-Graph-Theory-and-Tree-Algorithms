# Java Graph Theory and Tree Algorithms

This repository contains a comprehensive collection of exercises and projects demonstrating advanced proficiency in **Java 21**, focusing on **Functional Programming**, **Graph Theory (JGraphT)**, **Dynamic Programming**, and **Algorithm Complexity Analysis**.

##  Projects Overview

| Exercise Folder | Topic | Description |
| :--- | :--- | :--- |
| `ejercicio1-arboles` | **Recursive Types** | Processing and pathfinding in tree structures (Binary/N-ary). |
| `ejercicio2-biologia` | **Graph Theory** | Analyzing biological data using directed graphs (`SimpleDirectedGraph`). |
| `ejercicio3-optimizacion`| **Combinatorial Opt.**| Solving optimization problems (coloring, paths) using `JGraphT`. |
| `ejercicio4-grafos` | **Graph Algorithms** | Advanced pathfinding and subgraph visualization. |
| `ejercicio5-complejidad`| **Performance** | Benchmarking recursive vs iterative algorithms ($O(n)$ vs $O(2^n)$). |
| `extra-arboles-binarios`| **Binary Tree Logic** | Path optimization based on node constraints and depth. |

---

## Technology Stack
* **Language:** Java 21
* **Key Libraries:**
    * `JGraphT`: Used for graph structures, connectivity, and shortest path algorithms.
    * `LSI Utilities`: Custom library for input parsing (`GraphsReader`) and visualization (`GraphColors`).
* **Tools:** Maven, Git.
