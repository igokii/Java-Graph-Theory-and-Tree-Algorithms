# Exercise 2: Recursive Paradigms & List Processing

This project implements the same mathematical algorithm using four different programming paradigms to compare approaches.

## Algorithms
1.  **`f_it`**: Iterative implementation using a `while` loop.
2.  **`f_RNF`**: Non-final recursive implementation (results are built on the way back up the recursion stack).
3.  **`f_RF`**: Final recursive implementation (tail recursion with an accumulator to optimize stack usage).
4.  **`f_funcional`**: Implementation using Java Streams.
